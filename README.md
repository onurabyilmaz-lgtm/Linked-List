# Custom Linked List Implementation in C++

## Overview
This repository contains a from-scratch implementation of a Linked List data structure in C++. The primary goal of this project is to demonstrate a deep understanding of low-level memory management, pointer arithmetic, and Object-Oriented Programming (OOP) without relying on the C++ Standard Template Library (STL).

## Features
* Dynamic Node Management: Nodes are dynamically allocated and deallocated to ensure efficient memory usage.
* Core Operations:
  * Add: Inserts a new node into the list.
  * Delete: Safely removes a node and frees the associated memory to prevent memory leaks.
  * Search: Traverses the list via pointers to locate specific data.
  * Print: Iterates through the list and displays the current state.

## Technical Focus
* Language: C++
* Key Concepts: Raw Pointers, Dynamic Memory Allocation (new / delete), Memory Leak Prevention, Data Structure Architecture.

## How to Run
To compile and run this project from the terminal:

g++ main.cpp -o linkedlist
./linkedlist
